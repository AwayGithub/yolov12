## 一、项目概要与架构

**任务：** YOLOv12n + RGBT-3M，RGB+IR 双模态目标检测，nc=3（smoke/fire/person），分辨率 480×640。

**动机：** 早期融合（6ch 直接拼接）在第一层即混合两种物理特性截然不同的模态，低层特征提取缺乏模态针对性。改为中期融合双分支，让每个模态独立提取低层特征，再在高语义层级交互。

**决策：** 采用自定义 `DualStreamDetectionModel`（Python 层实现双分支，不修改 `parse_model()`，YAML 继续定义单分支结构）。

```
Input: (B, 6, H, W)
         │
    ┌────┴────┐  split: 0:3=RGB, 3:6=IR
    ▼         ▼
 RGB(B,3,H,W)  IR(B,3,H,W)
    │              │
┌───┴───┐    ┌────┴────┐
│ Conv  │    │  Conv'  │   Stage 0-3: P1/2 → P3/8
│ C3k2  │    │  C3k2'  │   Stage 4  ← P3 特征
│A2C2f/ │    │A2C2f'/  │   Stage 6  ← P4（可选 CMG/CMA）
│ C3k2  │    │ C3k2'   │
│ A2C2f │    │ A2C2f'  │   Stage 8  ← P5（可选 CMG/CMA）
└───┬───┘    └────┬────┘
    │              │
    ▼              ▼
 融合（P2/P3/P4/P5）：Concat+1×1Conv 或 DMGFusion(v1/v2)@P2
    │
    ▼
  Shared Neck (FPN) → Detect Head
        └─ 可选 RGB/IR 辅助头 @P3（仅训练）
```

**YAML 控制标志**（`ultralytics/cfg/models/v12/yolov12-dual*.yaml`）：

| 标志 | 作用 | 合法值 |
|------|------|--------|
| `cmg_stages` | SE-CMG 跨模态门控所在层级 | 子集 of `{p3,p4,p5}` |
| `cma_stages` | CrossModalA2C2f 双向跨模态注意力层级 | 子集 of `{p3,p4,p5}` |
| `p2_fusion` | P2 融合方式 | `plain` / `dmg` / `dmg_v2` |

## 二、核心模块规范

### 2.1 CrossModalGating（SE-CMG）

`ultralytics/nn/modules/block.py`。SE 式通道门控：GAP → Linear → Sigmoid → 残差。`guide` 模态的全局通道统计量调制 `target` 模态特征。每实例参数 = c×c + c。

```python
class CrossModalGating(nn.Module):
    def __init__(self, channels):
        super().__init__()
        self.pool = nn.AdaptiveAvgPool2d(1)
        self.fc = nn.Linear(channels, channels)

    def forward(self, target, guide):
        gate = torch.sigmoid(self.fc(self.pool(guide).flatten(1)))  # (B, C)
        gate = gate.unsqueeze(-1).unsqueeze(-1)
        return target * gate + target
```

> Exp-2b 验证 CM-CBAM（空间注意力）因 `torch.mean/max(dim=1)` 跨通道规约对 GPU cache 不友好，推理 +166%、指标劣于 SE-CMG，已否定。

### 2.2 CrossModalA2C2f（CMA）

**动机：** CMG 是后置模块，信息太晚、信号太粗（GAP 丢失空间位置）。CMA 将跨模态交互内嵌进 A2C2f：前半段自注意力（建立模态内理解），后半段用另一模态空间特征作 K/V。

**nano 配置（n=2）：** 1 组 2×ABlock 自注意力 + 1 组 CrossModalABlock 跨模态注意力。`cross_scale=0.01` 可学习残差缩放，保证初始化无害。

| 维度 | CMG | CMA |
|------|-----|-----|
| 信息粒度 | 全局 1 值/通道 | 每空间位置 Q-K 匹配 |
| 空间感知 | 无 | 有 |
| P5 参数/实例 | ~33K | ~377K |

### 2.3 DMGFusion v1（P2 融合模块）

差异引导融合：`fused = (1 + α·S)·(w_rgb·R + w_ir·I) + β·(R+I)/2`，其中 `D=|R-I|`，`S=σ(diff_enc(D))`，`[w_rgb,w_ir]=softmax(sel([R;I;D]))`。初始 α=0、β=1。

**设计缺陷（Exp-7 特征可视化证实，见 §6.2）：** β 残差吞噬门控；D 编码尺度差而非语义分歧；softmax 零和约束引发目标处"双低洞"；BN 导致输出幅值塌缩；S 整图饱和；W 无通道维。

### 2.4 DMGFusion v2

相对 v1 的逐项修复：

| 改动 | v1 | v2 |
|------|----|----|
| β 残差 | 保留 | **彻底移除** |
| S/α 分支 | 保留 | **彻底移除** |
| D 归一化 | 原始 `\|R−I\|` | `\|IN(R)−IN(I)\|`（InstanceNorm） |
| W 门类型 | softmax（零和） | 两路独立 sigmoid |
| W 维度 | (B,2,H,W) | 因式分解 (B,2,H,W)⊗(B,2,C,1,1) |
| out_proj 归一化 | BN | GroupNorm(8) |
| 融合公式 | 见 v1 | `out_proj(w_r·R + w_i·I)` |

加权求和时用原始 x_rgb/x_ir（保留 IR 绝对幅值），IN 仅用于计算 D 与喂 sel。

### 2.5 RGB/IR 辅助检测头（P3 单尺度）

`DualStreamDetectionModel.__init__` 创建 `aux_head_rgb` / `aux_head_ir`（`Detect(nc, ch=[c_p3])`, stride=8）。训练前向在 CMG 之前对两分支 pre-CMG 的 P3 特征分别预测，`DualStreamDetectionLoss` 把 `aux_weight·(L_rgb+L_ir)` 加到主损失（默认 `--aux_loss_weight 0.25`）。推理时 `self.training=False` → aux head 不参与。

**目的：** 切断 RGB backbone 语义坍塌（RGB 无独立梯度 → 退化为纹理提取器 → fused 被 RGB 草污染）。

| 类别 | 目标数 | 宽度中位 | 高度中位 | 短边<32px | 32–96px | >96px |

smoke mean=186 >> median=111，分布高度右偏（少量近距离大烟团）。

### 3.2 各尺度对小目标的覆盖（某 stride 下 <2 格子比例）

| 类别 | P2 (s=4) | P3 (s=8) | P4 (s=16) | P5 (s=32) |
|------|---------:|---------:|----------:|----------:|
| smoke  |  0.2% |  6.2% | 22.2% | 39.5% |
| fire   |  5.2% | **35.8%** | **80.8%** | **97.8%** |
| person |  5.0% | **38.0%** | **71.4%** | **98.7%** |

### 3.3 关键结论

- **P3 是 fire/person 主力**，但已有 35–38% 目标不足 2 格子。
- **P4/P5 对 fire/person 几乎无直接贡献**，其价值在于提供 FPN 语义上下文。
- **P2（stride=4）可将 fire/person 覆盖率 62%→95%**，代价 ~+58% FLOPs。
- **CMA@P5 的作用机制**：P5 弥散热信号作为全局热源先验，经 FPN top-down 流至 P3 提供语义上下文。**P4 CMA 有害** —— fire/person 在 P4 处于"半可见"边界，跨模态对齐噪声干扰。

## 五、阶段 I 发现（Exp-0~4c：双分支 + CMG/CMA）

### 5.1 双分支中期融合是核心贡献（Exp-0→Exp-1）

| 类别 | Exp-0 | Exp-1 | Δ |
|------|------:|------:|------:|
| smoke  | 0.734 | 0.735 | +0.1% |
| fire   | 0.545 | **0.575** | **+3.0%** |
| person | 0.537 | 0.559 | +2.2% |

早期融合给 fire 仅 +4.8%（vs RGB-only）；再加双分支又 +3.0%，累计 +15.7%。双分支释放的互补性是早期融合的 1.6 倍。后续 CMG/CMA 均是此基础上的精细化。

### 5.2 单模态基线：三类目标模态依赖模式不同

| 类别 | RGB-only | IR-only | 模态关系 |
|------|---------:|--------:|----------|
| smoke  | **0.735** | 0.664 | RGB 强主导 |
| fire   | 0.497 | 0.524 | **两模态等效，真正互补** |
| person | 0.515 | 0.504 | 两模态接近，微弱互补 |

fire 对融合架构最敏感 —— 两模态单独接近，但信息完全不同（RGB 颜色/形状 vs IR 热辐射）。

### 5.3 CMG 对 person 不可替代（Exp-2）

CMG 消融（Exp-4b-noCMG vs Exp-4b）：

| 类别 | ΔmAP50 | ΔRecall |
|------|-------:|--------:|
| smoke  | 0.0% | +0.2% |
| fire   | +0.4% | −0.1% |
| **person** | **+1.1%** | **+1.1%** |

person 同时依赖 RGB 纹理（轮廓）与 IR 热辐射（生命体确认），CMG 动态平衡"哪些通道的 IR 信号更可信"。smoke/fire 各有主导模态，CMG 反而引入投票噪声。

### 5.4 CMA 仅在 P5 有正效果（Exp-4 系列）

| 配置 | mAP50-95 | person | 推理(ms) |
|------|------:|------:|------:|
| Exp-4d (CMA@P4) | 0.623 | 0.559 | 16.68 |
| **Exp-4c (CMA@P5)** ★ | **0.632** | **0.573** | 16.05 |
| Exp-4b (CMA@P4P5) | 0.627 | 0.566 | 17.55 |

P4 CMA 对 person 反而有害（0.559 < Exp-2 基线 0.562）—— P4 语义未成熟。P4P5 组合不如 P5-only。

### 5.5 CMG+CMA@P5 对 person 存在协同（后在 §7 被推翻）

| 配置 | person | Δ vs Exp-1 |
|------|------:|------:|
| Exp-1 | 0.559 | — |
| Exp-2 (+CMG) | 0.562 | +0.003 |
| Exp-4b-noCMG (+CMA@P4P5) | 0.564 | +0.005 |
| **Exp-4c (CMG+CMA@P5)** | **0.573** | **+0.014** |

独立叠加预期 +0.008，实际 +0.014（1.75×）。当时的机制解释："CMG 在通道维选出可信通道 → CMA 的 K/V 更纯净；CMA 的空间上下文反哺 CMG 门控。"

> **⚠️ §7 修正：** 阶段 III 发现 aux head + C3k2 接入后，CMG/CMA 的贡献被同源更根本的修复吸收。此处协同效应的真正来源是"RGB 独立梯度缺失时的次优补偿"。详见 §7.3。

### 5.6 单向 CMA 只提高 Recall、不提升定位

| 指标 | Exp-2 | Exp-4a（单向） | Exp-4c（双向） |
|------|------:|------:|------:|
| all Recall | 0.894 | **0.900** | 0.892 |
| all mAP50-95 | 0.627 | 0.625 | **0.632** |
| person mAP50-95 | 0.562 | 0.562 | **0.573** |

定位精度需要**双向交互**。单向以接近双向的代价只得 Recall，不值得保留。

## 六、阶段 II 诊断（Exp-7：DMGFusion@P2 首次失败）

### 6.1 实验层面（Exp-7 的 2×2 析因设计）

| | +P2 head | 无P2 head | P2 head 纯效果 |
|---|---|---|---|
| **DMGFusion@P2** | 7a: 0.628 | 7c: 0.625 | +0.003 |
| **Concat@P2** | 7b: 0.634 | 7d: 0.634 | 0.000 |
| DMGFusion vs Concat | −0.006 | **−0.009** | |

**三条直接结论：**
1. **P2 检测头对 mAP50-95 零贡献**（7b vs 7d=0.634 持平），仅提升 mAP50 +0.7%、Recall +0.4%。代价 GFLOPs +43%、推理 +4%。
2. **DMGFusion v1 经 FPN 污染上层**（7c=0.625 < 4c=0.632，person −1.5% 受损最重）。
3. **Concat@P2 对 FPN 中性至有益**（7d=0.634 ≥ 4c=0.632，fire +0.7%）—— P2 融合点本身有价值，DMGFusion 失败源于模块设计。

### 6.2 特征可视化诊断（video10_frame_01154）

> 详细工具：`tools/visualize_p2_dmg.py` + `tools/probe_const_input.py`。

**A. RGB 支路语义坍塌（根本问题）**

- RGB P2/P3 以草/植被高频纹理为主，目标激活低于背景；P4 退化为噪声；P5 仅零散全局激活点。
- IR 在 P2–P5 每尺度清晰定位 fire/person（目标-背景对比 10:1+）。
- 融合后 IR 目标能量被同数量 RGB 通道"平均"稀释；plain fusion 的 1×1 Conv 无"信 IR"先验。
- **根因：** detection loss 梯度经 IR 强信号路回传，RGB backbone 无独立约束 → 退化为纹理提取器。
- IR 过曝条带（x_ir≈3.0）未被抑制，在 DMGFusion 的 D 中最亮，被 sel_net 误识为"高价值分歧区"。

**B. P4 area=4 结构性噪声（与数据无关，zero-input probe 证实）**

- P4 mean row_std/col_std ratio = **2.29**，29% 通道 ratio ≥ 2.0。
- 原因：AAttn 把 H 切成 4 条 strip，各 strip token 数不同 → softmax 温度差异 → strip 间均值偏移。
- CMG@P4 作用于已污染特征 → 放大而非修复。
- P5 area=1 即全局注意力，ratio=1.98；bot[b05]=9.26 为学到的位置先验（烟雾偏上、人偏下），属有语义的数据集先验，不需修复。

**C. DMGFusion v1 六项设计缺陷**

1. **β 残差旁路吞噬门控（最致命）**：初始 α=0,β=1 → fused ≈ out_proj(0.5·(R+I))，W 合理但 fused 被 RGB 草主导。
2. **D=|R−I| 编码尺度差而非语义分歧**：x_rgb~[0.25,1.75]、x_ir~[0,3.0]，D 最亮在 IR 过曝条带而非目标。修复：InstanceNorm 前归一化。
3. **softmax 双路权重零和**：强制 w_rgb+w_ir≡1。目标位置两路同时出现"双低洞"（~0.3–0.4）。修复：独立 sigmoid。
4. **输出幅值塌缩 5–10 倍**：输入 ~[0,3]，fused ~[-0.6,0.2]。out_proj 含 BN → 零均值强制。修复：GroupNorm(8)。
5. **S 整图饱和失去空间选择性**：D 处处非零 → σ(diff_enc(D)) 全图接近饱和，α 只能均匀放大。修复：去 S/α。
6. **W 无通道维**：(B,2,H,W) 所有通道共享空间权重；不同通道（边缘/纹理/颜色/温度）需不同模态偏好。修复：因式分解 W_s ⊗ W_c。

### 7.3 CMG/CMA 冗余假说（Exp-8a vs Exp-8d）

**核心观察：** Exp-8d 反超 Exp-8a +0.6%，且 Params −21%（5.55M→4.37M）。这是本阶段最反直觉的信号：**Step 1+2 修复到位后，CMG/CMA 从"主力贡献者"退化为"冗余模块"**。

**机制解释：** Exp-4c 基线下 CMG/CMA 提供了两件事：
1. 跨模态通道重加权（SE-CMG）—— 修正 RGB backbone 语义塌缩；
2. 跨尺度注意力交换（双向 CMA@P5）—— 用 IR 先验稀释 P5 条带噪声。

Step 1（aux head）从**源头**给 RGB 独立梯度 → 解决 (1) 的病灶；Step 2（P4 C3k2）切断条带噪声传播路径 → 消除 (2) 的发生条件。两个前序修复到位后，CMG/CMA 再做同样工作即重复劳动，且引入参数与 BN 统计的训练噪声。

**逐类验证：** person 提升最大（+1.2% from 0.581→0.593），而 person 原本是 §5.5 认为"最依赖 CMG+CMA 协同"的类。此结果**修正 §5.5**：真正起作用的是 RGB 独立梯度本身，CMG/CMA 仅是缺失该梯度时的次优补偿。

### 7.4 DMG v1 / v2 / plain 三方消融（干净基线 Exp-8d/8e/8f）

所有三实验共享 P4 C3k2 + P3 aux + 无 CMG/CMA + 4-scale head，唯一变量 = P2 融合方式。

| 实验 | P2 融合 | all mAP50-95 | smoke | fire | person | ΔmAP50-95 vs plain |
|------|---------|------:|------:|------:|------:|------:|
| Exp-8e | plain Concat+Conv1×1 | 0.639 | 0.739 | 0.594 | 0.585 | — |
| Exp-8f | DMG v2 | 0.641 | 0.740 | 0.595 | 0.589 | **+0.002** |
| **Exp-8d ★** | **DMG v1** | **0.644** | 0.739 | **0.599** | **0.593** | **+0.005** |

**三点证据链：**

1. **单调排序 plain < v2 < v1**，间距均 >0.3% 判定阈值，排除噪声。三类 mAP50-95 方向一致（smoke 持平；fire 与 person 均随档位上升）。
2. **v2 的有限收益**（+0.002）与 §7.5.3 viz 结论吻合：v2 六项改动中仅 `sigmoid + IN(D)` 两项小幅正作用，删除 α·S 等于移除 v1 的主工作机制（Exp-8a α=+0.999 饱和）。v2 在干净基线下凭借 sigmoid/IN 细节弥补一部分，但无法弥补核心机制缺失。
3. **v1 的独立增益**（+0.005）在干净基线上仍被保留，结合 §7.5.2 的跨基线 α 符号翻转证据，**DMG v1 作为融合算子的有效性获得两条独立证据链支持**。

#### 7.5.2 DMGFusion v1 @P2 收敛标量（跨基线对比）

| 实验 | 上游基线 | α | β | 工作状态 |
|------|---------|---:|---:|------|
| Exp-7a | 无 aux + P4 A2C2f | **−1.091** | **+0.201** | α 负 → `(1+αS)` 在 S→1 区域近 0 → 模型主动**抑制**差异调制 → 门控未启用 |
| Exp-8a | +aux + P4 C3k2 + CMG/CMA | **+0.999** | **+0.247** | α 接近饱和正值 → 分歧区放大 ~2× → 差异门控**主导**前向，β 仍为正残差骨架 |
| **Exp-8d** ★ | +aux + P4 C3k2 + 无 CMG/CMA | **+2.420** | **−0.202** | α 越过 +1 饱和进入"超放大"区（S→1 处 fused 放大 ~3.4×）；**β 翻为负值** → `(R+I)/2` 由残差锚定升级为**共模参考被主动减去** |

**演化轨迹（ADR-001 核心证据之一）：**

随上游基线由脏到净，α 单调上行（−1.091 → +0.999 → +2.420），β 由正转负（+0.201 → +0.247 → **−0.202**）。**一个 1 维可学参数（β）在训练中翻符号，意味着 `(R+I)/2` 路径的物理角色发生了质变**：

- Exp-7a/8a 时 β>0：`(R+I)/2` 作为"均值残差"提供低频骨架保底；
- **Exp-8d 时 β<0：`(R+I)/2` 升级为"共模参考"被主动减去**，形成 `(modality-selected enhancement) − β·(common-mode baseline)` 的**差分放大 + 共模抑制**结构。

即 Exp-8d 最终形态可改写为：
```
fused ≈ (1 + 2.42·S)·(w_r·R + w_i·I) − 0.10·(R+I) + out_proj 偏置
        └─── 差分放大（分歧区主导）───┘   └─ 共模抑制 ─┘
```

这是典型的 differential amplifier 工作模式：**在模态分歧像素（目标）处强化加权和，同时扣除两模态共有的背景基线**。α=+0.999→+2.420 表示差分增益继续增长；β 翻负是这一演化的下一步自发产物——模型发现"主动消去共模"比"被动叠加均值"更利于目标判别。

**结论：** v1 的机制有效性由三点轨迹证据（两次饱和 + 一次符号翻转）支撑，远比单一消融表格更直接——这不是"模块恰好有效"，而是一个连续参数在干净基线下**朝物理合理方向自我演化**。Step 1+2+3（去 CMG/CMA）的清障顺序与参数演化顺序完全一致，验证"先清障再创新"的技术路线。

**三样本 viz（video4_frame_01746 / video3_frame_02430 / video9_frame_00149）观察：**

| 组件 | 行为 | 判定 |
|------|------|------|
| `r_n`/`i_n` (IN 后) | 两模态通道分化显著，IR 热源尖锐、RGB 提供几何轮廓 | 正常 |
| `D = \|r_n − i_n\|` | 分歧区对齐目标（tripod/人体/火焰） | 信号可用，但 v2 未消费（α·S 已删） |

检出质量与 W_s 清晰度正相关：frame_00149（W_s 反向互补最明显）fire/person conf=0.84；frame_02430（W_s 模糊）person conf=0.00。

**v2 六项改动逐项归因：**

| v2 改动 | 实测 | 归因 |
|---------|------|------|
| 删除 β 残差 | 负作用 | v1 中 β=0.247 是稳定低频骨架，非吞噬项 |
| 删除 α·S 差异调制 | **负作用（关键）** | v1 Exp-8a α=+0.999 实际主导前向，v2 切除后 D 无从消费 |
| IN → D / sel | 小幅正 | 原始 `\|R−I\|` 被 BN 通道方差污染；IN 后语义对齐改善 |
| softmax → 独立 sigmoid | 小幅正 | W_s 能形成反向互补；代价是丢失模态竞争先验 |
| W 因式分解 (H,W)⊗(C) | **无效（核心拖累）** | W_c 塌缩至 ≈0.5，未学到通道路由，反压 W_s 幅值 |
| BN → GroupNorm(8) | 中性 | Batch=16、C=64 下 BN 已稳定 |

**核心结论：** v2 删掉了 v1 的"功"（α·S），保留的是"形"（W 门控）。v1 的 α→+0.999 饱和是差异门控真正工作的直接证据；v2 在干净基线（Exp-8f=0.641）能逼近 v1（0.644）主要靠 `sigmoid + IN(D)` 的小幅修正弥补了去 α·S 的损失一半。

#### 7.5.4 设计教训

1. **不要在一次实验里同时改 ≥3 个子模块。** v2 六项同步修改导致归因困难。
2. **SE-style 通道门在极低语义层（P2）基本无效。** GAP 把 P2 真正编码的空间差异信号平均掉，后续若需通道门应在 P4/P5 使用。
3. **`|α|→1` 饱和是强信号而非故障。** 应视为"该机制在承担主要工作"，不要轻易删除。

## 八、最终判定与移交

**阶段 III 主方案（冻结）：Exp-8d**
- DMGFusion v1 @P2 + P4 C3k2 backbone + P3 aux head（权重 0.25）+ 无 CMG/CMA + 4-scale head
- **mAP50-95 = 0.644**（fire 0.599 / person 0.593 / smoke 0.739）
- 4.37M params / 31.56 GFLOPs

**阶段遗留问题（移交 ADR-002）：**

1. **DMG v1 内部存在明显可优化点。** §7.5.3 viz 已指出：β 残差与 α·S 调制路径在 Exp-8a 已进入非平凡工作点，Exp-8d 进一步演化为 α=+2.420 / β=−0.202 的"差分放大 + 共模抑制"形态（§7.5.2）——核心机制确实在工作且仍未饱和上限。但 softmax 零和、W 无通道维、BN 幅值塌缩等缺陷未在 Exp-8d 中解决。v1→v2 的失败在于重构策略错误（大改而非定向微调）。下阶段应在 v1 基础上只替换 softmax→sigmoid + 前置 IN(D)，保留 α·S+β。
2. **P2 融合模块增益天花板较低**（plain→v1 仅 +0.005）。若追求更大提升，应考虑 P2 aux head 或监督层面改造，而非 P2 模块雕花。
3. **CMG/CMA 已完成历史使命。** 阶段 I 诊断工具兼治疗方案，阶段 III 已退役。未来新基线不再包含。
